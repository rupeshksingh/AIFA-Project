Disaster-response logistics planning

Symbolic STRIPS-style operators over roads, bulldozers, and medical teams—solved with BFS, UCS, GBFS, and A* using domain heuristics.

Run the demo Repository README Project report (PDF)

Domain & formulation

State

Immutable facts: connectivity, road status (clear / blocked), resource positions, and victim treatment status at each location.

Goal

Required victim sites must be victims_treated. The planner searches until all goal facts hold.

Operators

  • Move(resource, from, to) — travel along a clear edge.
  • Clear(bulldozer, from, to) — open a blocked road segment.
  • Treat(med_team, location) — treat victims at a site.

Implemented techniques

BFS

Uninformed breadth-first search.

UCS

Uniform cost (unit step cost in this domain).

GBFS

Greedy best-first on a selectable heuristic.

A*

f = path cost + heuristic; admissible options include victim count.

Heuristics: zero, untreated_victims, blocked_roads, hybrid_response.
